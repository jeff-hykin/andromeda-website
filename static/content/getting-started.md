# Getting Started with Andromeda

To get started with Andromeda, follow these steps:

1. **Install**
   ```bash
   cargo install --git https://github.com/tryandromeda/andromeda
   ```

2. **Run**
   ```bash
   andromeda run example.ts
   ```
